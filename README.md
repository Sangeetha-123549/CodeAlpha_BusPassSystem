# 🚍 CodeAlpha BusPassSystem

## 📌 Project Title
Cloud-Based Bus Pass / Ticket Booking System

---

## 🎯 Objective
To develop a secure, scalable, cloud-based bus ticket booking system that prevents ticket loss, theft, and incorrect pricing.

---

## ⚙️ Features
- User Registration & Login
- Secure Session Management
- Ticket Booking System
- Auto Price Calculation (₹50 per ticket)
- Unique Ticket ID Generation
- Booking Confirmation Page
- Cloud Deployment Ready

---

## 🛠 Tech Stack
- Frontend: HTML, CSS
- Backend: Python Flask
- Database: SQLite
- Cloud: Render / Railway
